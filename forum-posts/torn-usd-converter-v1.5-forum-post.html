Torn USD Converter v1.5
UPDATE // RECONVERSION FIX + MIL/BIL SUFFIXES
v1.5 is a bugfix and small feature update. The big one: values that were already converted to USD no longer get re-overwritten by TornTools or chat mutations.
Uses the pawn shop donator pack conversion rate of $0.000001 (§5) USD / 23,500,000
What's New
  • > Reconversion fix — already-converted text nodes are now skipped via a data-usdConverted flag on parent elements and (§ detection in text content, so TornTools/chat can't re-override USD values
  • > mil / bil suffix support — the value regex now recognises "mil" and "bil" alongside the existing k/M/B/T suffixes (case-insensitive), so $23.5mil converts correctly
  • > MutationObserver guard — the isAddedNodes guard ensures the script's own DOM changes don't trigger a re-processing loop
  • > Parent element marking — processNode now marks the parent element after conversion so subsequent mutations skip already-processed nodes
